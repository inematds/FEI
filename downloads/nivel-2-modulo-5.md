# Modulo 5: Exercicios Progressivos
## Nivel 2 - Programa Completo

**FEI - Formacao em Engenharia de Intencao**

## Introducao

Este e o modulo mais importante para sua transformacao: de executor para **designer de intencao**.

Voce fara **12 exercicios progressivos** que evoluem sua habilidade de forma gradual e sistematica.

### Objetivo do Modulo

Transformar conhecimento teorico em habilidade pratica. Ao final, voce sera capaz de criar intencoes precisas para qualquer contexto.

## Exercicio 1: Converter Prompts Antigos

### Objetivo

Identificar elementos ultrapassados e transformar em Engenharia de Intencao.

### Tarefa

Converta 5 prompts antigos em intencoes modernas:

**Exemplo:**

**Prompt Antigo:**
```
Voce e um especialista em SEO com 10 anos de experiencia.
Liste 10 estrategias de SEO para e-commerce.
Use bullet points.
Seja objetivo.
```

**Conversao para Engenharia de Intencao:**
```
Objetivo: Identificar 10 estrategias de SEO para e-commerce

Intencao:
Para que: aumentar trafego organico em loja online
Impacto: implementacao pratica nos proximos 30 dias

Delegacao:
Me pergunte sobre o nicho, tamanho da loja e recursos disponiveis
```

### Exercicios para Voce Fazer

Converta estes 5 prompts:

1. "Crie um titulo chamativo para meu post de blog"
2. "Explique Machine Learning de forma simples"
3. "Faca um resumo deste texto em 3 paragrafos"
4. "De ideias de conteudo para Instagram"
5. "Crie uma introducao para meu e-book"

## Exercicio 2: Construir Intencao Completa do Zero

### Objetivo

Praticar a estrutura completa das 7 etapas.

### Tarefa

Escolha uma situacao real da sua vida profissional e construa uma intencao completa.

**Template:**

```
1. OBJETIVO
[Acao] + [Artefato] + [Escopo]

2. INTENCAO
Para quem: [audiencia]
Para que: [proposito]
Impacto: [resultado esperado]

3. CRITERIOS
- Tom: [definir]
- Profundidade: [definir]
- Estilo: [definir]

4. CONTEXTO
Situacao atual: [descrever]
Dados relevantes: [listar]
Restricoes: [listar]

5. RACIOCINIO (opcional)
[tipo de raciocinio necessario]

6. DELEGACAO
Antes de responder, pergunte:
- [pergunta 1]
- [pergunta 2]

7. FORMATO (opcional)
[estrutura especifica]
```

### Validacao

Sua intencao deve:
- [ ] Ter objetivo mensuravel
- [ ] Definir audiencia especifica
- [ ] Incluir contexto relevante (nao excessivo)
- [ ] Delegar ao menos 2 perguntas
- [ ] Especificar criterios claros

## Exercicio 3: Ensinar IA a Perguntar

### Objetivo

Dominar delegacao inteligente.

### Tarefa

Para cada cenario, crie a secao de delegacao que faz a IA perguntar ANTES de responder.

**Exemplo:**

**Cenario:** Criar plano de estudos

**Delegacao:**
```
Antes de criar o plano, me pergunte:
- Qual seu nivel atual de conhecimento?
- Quantas horas por semana voce tem disponiveis?
- Qual o objetivo final (certificacao, trabalho, interesse)?
- Prefere teoria primeiro ou pratica desde o inicio?
```

### Cenarios para Praticar

1. Criar campanha de e-mail marketing
2. Desenvolver estrategia de vendas
3. Montar curriculo profissional
4. Planejar projeto de software
5. Criar conteudo educacional

## Exercicio 4: Criterios de Qualidade

### Objetivo

Aprender a especificar "como avaliar se e bom".

### Tarefa

Para cada situacao, defina criterios de qualidade apropriados.

**Exemplo:**

**Situacao:** E-mail para investidor

**Criterios:**
```
- Tom: profissional mas confiante (nao suplicante)
- Estrutura: direto ao ponto (investidores tem pouco tempo)
- Dados: quantitativos e verificaveis
- Foco: traction e visao, nao apenas ideia
```

### Situacoes para Praticar

1. Post para LinkedIn corporativo
2. Video educacional para YouTube
3. Apresentacao para equipe tecnica
4. Artigo de blog para iniciantes
5. Proposta comercial B2B

## Exercicio 5: Raciocinio Critico vs Criativo vs Tecnico

### Objetivo

Identificar qual tipo de raciocinio e adequado para cada situacao.

### Tarefa

Classifique e justifique:

**Exemplo:**

**Situacao:** Decidir entre duas tecnologias para projeto

**Tipo:** Raciocinio Critico

**Justificativa:** Precisa avaliar pros, contras, riscos e tomar decisao informada

### Situacoes para Classificar

1. Criar nome para startup
2. Analisar viabilidade de negocio
3. Debugar codigo com erro
4. Desenvolver campanha publicitaria
5. Otimizar processo operacional
6. Escrever roteiro de filme
7. Avaliar proposta de parceria
8. Arquitetar sistema de software

---

## Exercicio 6: Melhoria Iterativa com Revisao Guiada

### Objetivo

Aprender a refinar respostas da IA com feedback estruturado.

### Tarefa

1. Crie uma intencao inicial (pode ser imperfeita)
2. Receba resposta da IA
3. Identifique o que faltou
4. Refine a intencao
5. Compare resultados

**Framework de Refinamento:**

```
O que funcionou bem:
- [aspecto 1]
- [aspecto 2]

O que faltou:
- [lacuna 1]
- [lacuna 2]

Intencao refinada:
[versao melhorada]
```

## Exercicio 7: Refinar Objetivos Ambiguos

### Objetivo

Transformar pedidos vagos em objetivos claros.

### Tarefa

Refine estes objetivos ambiguos:

**Exemplo:**

**Ambiguo:** "Melhorar meu site"

**Refinado:** "Aumentar taxa de conversao do site de 1.5% para 3% em 60 dias"

### Objetivos para Refinar

1. "Aumentar vendas"
2. "Melhorar produtividade"
3. "Fazer marketing"
4. "Aprender programacao"
5. "Criar conteudo"
6. "Otimizar processo"

## Exercicio 8: Intencoes para Tarefas Complexas

### Objetivo

Lidar com projetos multi-etapas.

### Tarefa

**Projeto:** Lancamento de produto digital completo (do zero ao mercado)

## Exercicio 9: Intencoes para Tarefas Pequenas

### Objetivo

Nao complicar o simples.

### Tarefa

Para tarefas simples, crie intencoes enxutas:

**Exemplo:**

**Tarefa:** Escrever tweet sobre lancamento de feature

**Intencao Enxuta:**
```
Tweet anunciando [feature] para [audiencia]
Tom: empolgado mas profissional
Inclua beneficio claro
```

### Tarefas para Praticar

1. Responder e-mail de agradecimento
2. Criar legenda para foto no Instagram
3. Resumir reuniao em 3 bullets
4. Sugerir titulo para apresentacao
5. Criar CTA para landing page

## Exercicio 10: Intencao para Contexto Emocional

### Objetivo

Lidar com situacoes delicadas.

### Tarefa

Crie intencoes para contextos que exigem sensibilidade:

**Exemplo:**

**Contexto:** E-mail comunicando demissao de funcionario (decisao da empresa por corte de custos)

**Intencao:**
```
Objetivo: Criar e-mail comunicando desligamento

Intencao:
Para: funcionario com 3 anos de empresa (bom relacionamento)
Para que: comunicar decisao com respeito e clareza
Impacto: que entenda que nao foi por desempenho

Criterios:
- Tom: honesto mas humano
- Reconhecer contribuicoes
- Oferecer suporte na transicao
- Nao usar eufemismos corporativos vazios
```

### Contextos para Praticar

1. Feedback critico para membro da equipe
2. Comunicado de atraso em projeto
3. Pedido de desculpas a cliente
4. Mensagem de luto/condolencias

## Exercicio 11: Intencao para Analise Profunda

### Objetivo

Pesquisa e analise complexa.

### Tarefa

Crie intencao para analise de:

**Tema:** Impacto da IA no mercado de trabalho nos proximos 10 anos

Sua intencao deve:
- Definir escopo geografico e setorial
- Especificar tipo de evidencia desejada
- Incluir delegacao sobre foco da analise
- Definir profundidade e angulo

---

## Exercicio 12: Intencao para Decisoes Estrategicas

### Objetivo

Usar IA para decisoes criticas.

### Tarefa

Crie intencao para decidir:

**Decisao:** Pivotar modelo de negocio ou dobrar aposta no atual?

Sua intencao deve:
- Solicitar analise multi-dimensional
- Incluir contexto completo
- Pedir cenarios (melhor/pior/provavel)
- Delegar perguntas sobre valores e visao
- Considerar reversibilidade da decisao
